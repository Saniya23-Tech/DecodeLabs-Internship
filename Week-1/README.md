# Week 1 - Rule-Based Chatbot 🤖

## 📌 Description
This project is a simple rule-based chatbot that responds to user inputs based on predefined conditions. It does not use machine learning or AI models; instead, it follows a set of if-else rules to generate responses.

Through this project, I understood how basic chatbot logic works and how user input can be mapped to predefined outputs.

---

## 🧠 What I Learned
- How rule-based systems work
- Handling user input in Python
- Using conditional statements (if-else)
- Basic logic building for chat systems
